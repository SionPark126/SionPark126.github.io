GEOQUIZ

Geoquiz project This quiz is developed for a professor at Seattle Pacific University to facilitate learning expereince for her students. The quiz provides the maps of six continents, in which 10 random countries are colored for users to write its country and capital name.

Key Features:

- Clicking numbers will zoom into its corresponding country.
- Some possible misspellings get partial points.

Tools and Resources Used:

Javascript, HTML, CSS, Digital Ocean, MongoDB, Openlayers API, Canvas API


LETTER RUN

Letter Run project This is my senior software engineering project that I worked on with 3 other team members. We used Phaser.js and Electron to create a 2D platformer game. My responsibilities were implementing the game mechanics, adding assets to the game and creating stages.

Key Features:

- Player controls the character with arrow keys to collect letters displayed on the letter board.
- There are total 9 stages and 9 different words to complete.
- Player can choose different characters.

Tools and Resources Used:

Javascript, HTML, CSS, Electron, Phaser.js
