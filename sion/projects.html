GEOQUIZ

Geoquiz project This quiz is developed for a professor at Seattle Pacific University to facilitate learning expereince for her students. The quiz provides the maps of six continents, in which 10 random countries are colored for users to write its country and capital name.

Key Features:

- Clicking numbers will zoom into its corresponding country.
- Some possible misspellings get partial points.

Tools and Resources Used:

Javascript, HTML, CSS, Digital Ocean, MongoDB, Openlayers API, Canvas API
